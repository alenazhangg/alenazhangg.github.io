Data Structures and Algorithms Class Project

  • Programmed in Java to create a Monte Carlo simulation to estimate the value of a percolation threshold in a random system
    • Modeled a percolation system using an NxN grid of sites, where each site is either open or blocked
    • Full sites are open sites that can be connected to an open site at the top of the grid
    • The system is said to percolate if there is a full site in the bottom row
  • Implemented breadth-first-search, depth-first-search, and union-find to determine when a system percolates
  • Estimated the percolation threshold to be around 0.593 for a NxN grid by running repeated simulations
